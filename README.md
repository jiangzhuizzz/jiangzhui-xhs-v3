# jiangzhui-xhs-v2

小红书 Agent v2 - 基于大语言模型的智能小红书内容创作助手

## 项目简介

这是一个用于小红书内容创作的 AI Agent 项目，能够帮助用户生成高质量的小红书笔记内容。

## 项目结构

```
.
├── README.md           # 项目说明
├── SOUL.md             # Agent 系统提示词
├── requirements.txt    # Python 依赖
└── src/
    ├── agents/         # Agent 定义
    ├── tools/          # 工具集成
    └── workflows/      # 工作流
```

## 快速开始

### 安装依赖

```bash
pip install -r requirements.txt
```

## 功能特性

- 智能内容创作
- 小红书风格适配
- 多场景内容生成
